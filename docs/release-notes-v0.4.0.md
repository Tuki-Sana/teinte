## リリース本文（ここからコピー）

ローカルで画像を開き、解像度・EXIF・支配色やコントラストなどを確認できるデスクトップ向けツールです。**画像データを外部サーバーに送りません。** スポイトパレット、分析／パレット JSON、PDF 書き出し、用語集に対応しています。

### 0.3 からの変更

- **macOS 向けに Intel（x64）用の `.dmg` を追加**しました。M 系 Mac は **`aarch64`**、Intel Mac は **`x64`** が付いた名前のファイルを選んでください。
- **分析 JSON のスキーマ版・パレットの保存形式・解析の意味づけに変更はありません。** 0.3 で保存したデータはそのまま扱えます。

### ダウンロードの選び方

| 環境 | 選ぶアセットの目安 |
|------|-------------------|
| Mac（Apple Silicon） | 名前に **`aarch64`** の `.dmg` または `.app.tar.gz` |
| Mac（Intel） | 名前に **`x64`** の `.dmg` |
| Windows（64 ビット） | **`x64-setup.exe`** または **`x64_en-US.msi`** |

### macOS で「壊れている」または「開発元を確認できません」と表示されるとき

未署名・未公証のため、初回起動時にブロックされることがあります。ターミナルで以下を実行してから起動してください。

```bash
xattr -d com.apple.quarantine /Applications/Teinte.app
```

または**アプリを右クリック →「開く」**で起動できる場合もあります。
